# 🍜 中科大隨機食物選擇器 (NUTC Food Picker)

[![GitHub Pages](https://img.shields.io/badge/Deployed%20on-GitHub%20Pages-brightgreen)](https://chaworld.github.io/nutc_fd/)
[![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/ISC)

這是一個專為 **國立臺中科技大學 (NUTC)** 學生設計的隨機食物選擇器。為了解決「今天要在三民校區吃什麼？」或「民生校區附近有什麼好吃的？」這類每日難題，本專案提供了一個直觀且趣味的介面來幫助你做決定。

## 🌐 線上展示

立即體驗：[https://chaworld.github.io/nutc_fd/](https://chaworld.github.io/nutc_fd/)

## ✨ 功能特色

- **校區切換**：支援 **三民校區** 與 **民生校區** 周邊美食數據切換。
- **隨機抽取**：一鍵按下，從精選的口袋名單中隨機抽取今天的餐點。
- **精選名單**：收集了中科大學生常吃的在地美食、小吃與便當。
- **靈感功能**：透過 API 獲取額外的用餐靈感。
- **響應式設計**：支援手機與電腦瀏覽，隨時隨地都能選。

## 🛠️ 技術棧

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **API**: Vercel Serverless Functions (Node.js)
- **Deployment**: GitHub Pages

## 📂 專案結構

- `/docs`: 包含主要的網頁原始碼 (`index.html`, `style.css`, `script.js`) 與食物數據 (`data.js`)。
- `/api`: Vercel Serverless Functions 程式碼。
- `script.js`: 核心隨機邏輯處理。

## 🚀 如何使用

1. 開啟連結：[NUTC Food Picker](https://chaworld.github.io/nutc_fd/)
2. 選擇你的校區。
3. 點擊「點我選食物」按鈕。
4. 得到結果，出發去吃飯！
